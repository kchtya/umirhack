# Tetry Frontend

Frontend-часть платформы Tetry, отвечающая за визуальный конструктор лендингов. Реализована на Vue.js 3 и предоставляет пользователю интуитивный интерфейс для создания одностраничных сайтов методом перетаскивания готовых блоков.

## Архитектура и основные модули

Центральный модуль системы, обеспечивающий:
- Drag-and-drop сборку лендингов из готовых блоков
- Редактирование контента в реальном времени
- Визуальную настройку стилей и расположения элементов

src/:
-  components/ # Библиотека UI-компонентов и блоков 
-  views/ # Страницы редактора и превью 
-  router/ # Навигация между разделами  
-  stores/ # Управление состоянием приложения 
-  assets/ # Графические ресурсы и стили 
-  App.vue # Корневой компонент 

## Быстрый старт
### 1. Учтановка взаимосвязей
```bash
npm install
```
### 2. Запуск в режиме разработки
```bash
npm run dev
```
### 3. Production-сборка
```bash
npm run build
```
Приложение доступно по адресу: http://localhost:5173
