# Tetry Frontend

Frontend-часть платформы Tetry, отвечающая за визуальный конструктор лендингов. Реализована на Vue.js 3 и предоставляет пользователю интуитивный интерфейс для создания одностраничных сайтов методом перетаскивания готовых блоков. Данная папка содержит всю фронтенд-составляющую проекта Tetry - пользовательский интерфейс конструктора лендингов, систему визуального редактирования и все клиентские компоненты, обеспечивающие работу с конструктором в браузере.

## Архитектура и основные модули

### Визуальный редактор
Центральный модуль системы, обеспечивающий:
- Drag-and-drop сборку лендингов из готовых блоков
- Редактирование контента в реальном времени
- Визуальную настройку стилей и расположения элементов

### Компонентная структура
- **components/** - Библиотека UI-компонентов и блоков конструктора
- **views/** - Страницы редактора и режима предпросмотра
- **router/** - Навигация между разделами приложения
- **stores/** - Централизованное управление состоянием приложения
- **assets/** - Графические ресурсы и стилевое оформление
