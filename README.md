# Представляем вашему вниманию веб-приложение Tetry, разработанное командой "Слепой ковбой хромая лошадь" 

## О проекте

**Tetry** — это интуитивно понятный конструктор лендингов, созданный с душой для тех, кто хочет быстро запустить свой сайт без технических сложностей. 

Мы верим, что создание красивых сайтов должно быть доступно каждому!

### Что мы делаем особенного?
- **Визуальный редактор** — перетаскивайте блоки как в LEGO
- **Мгновенный результат** — видите изменения в реальном времени  
- **Адаптивный дизайн** — ваш сайт будет отлично смотреться на любом устройстве
- **Простота** — никаких сложных настроек и обучения

### Что где лежит:
- **frontend** - весь интерфейс конструктора на Vue.js
- **backend** - серверная часть и API (в разработке)
- **README.md** - этот файл с описанием проекта
- прочие файлы и папки, в которых содержится код или другой вид данных, который нужен для проекта

## Быстрый старт

### Всего 3 шага до запуска:

1. **Открываем репозиторий через Visual Studio Code**

2. **Переходим в папку фронтенда**
```bash
   cd frontend
```

3. **Устанавливаем зависимости**
  ```bash
    npm install
  ```

4. **Запускаем проект**
  ```bash
    npm run dev
  ```

Готово! Открывайте [http://localhost:5173](http://192.168.56.1:5173/) и начинайте творить!
