# Tetry Backend

Backend-часть платформы Tetry, отвечающая за работоспособность конструктора лендингов. Данная папка содержит всю бэкенд-составляющую проекта Tetry - серверную логику, API для взаимодействия с фронтендом, систему хранения данных и бизнес-логику конструктора лендингов. Обеспечивает работу с пользовательскими проектами, аутентификацию и управление данными.

## Архитектура

Backend построен на современном стеке технологий:
- **Node.js + Express** - серверная логика и API
- **Supabase** - платформа для хранения данных (PostgreSQL)
- **REST API** - взаимодействие с фронтендом
- **JWT Auth** - система аутентификации

## Основные модули

### API Эндпоинты
- Проекты пользователей (создание, редактирование, сохранение)
- Система аутентификации и авторизации  
- Управление блоками и шаблонами
- Генерация и выгрузка готовых лендингов

### База данных
- Хранение структур проектов в JSON формате
- Пользовательские данные и настройки
- Библиотека блоков и шаблонов
- Система прав доступа

## Быстрый старт
### 1. Установка зависимостей
```bash
npm install
```

### 2. Запуск в режиме разработки
```bash
npm run dev
```
### 3. Запуск продакшн версии
```bash
npm start
```
